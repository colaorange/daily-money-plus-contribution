概念

觀看影片版本: 新手記帳概念介紹
每日記帳本的主要概念很簡單, 只有兩個
  1. 各種不同的帳戶(依你的需要而定).
  2. 錢在在不同帳戶之間的轉換(你必需為每一次的交易,花費作記錄).

帳戶

共有五種不同型態的帳戶, 你可以在帳戶管理的頁面新增/編輯或刪除

  1. 收入 : 薪水..等
  2. 支出 : 食物,娛樂..等
  3. 資產 : 現金, 銀行..等
  4. 債務 : 信用卡, 貸款..等
  5. 其他 : ...
* 帳戶的排序是依型態及名字的順序,你可以在帳戶名字上加上1或A等前置字來排序你的帳戶.
* 帳戶名字可以用.(點)來分隔(例如 食.早餐,食.午餐), 明細編輯時, 將會提供較佳的階層式選擇.

轉換明細

當你的錢發生了轉換(例如你花了200去吃魯肉飯), 你使用主桌面的新增明細上的功能來建立一筆新的資料

  1. 選擇來源帳戶 : 現金
  2. 選擇目地帳戶 : 食物
  3. 選擇日期 : 今天
  4. 輸入金額 : 200
  5. 輸入描述 : 魯肉飯
  6. 點選建立，然後點選關閉(假設只需建立一筆)
上面只是一個非常簡單的例子, 想像一下, 如果你的薪水是直接匯入銀行 或是你在PHCOME上用信用卡買東西，本日記帳幫都可以幫您作記錄。你可以在上面的五種型態的帳戶類型中，任意建立你需要的帳戶.

使用範例

薪水直接到銀行

從帳戶 : 薪水
到帳戶 : 銀行

從帳戶 : 銀行

結算報告

時間區間的結算報告

結算報告是使用你建立的資料來統計計算出來的，並依本周,本月,本年來呈現。你可以點選報告的上方按鈕來改變時間的位置

累加的結算報告

當你想知道你目前有多少現金, 目前有多少資產, 而不是這個月花多少的時候，你會需要使用累加的結算報告。要讓累加的結果報告正確計算， 你必需設定每個帳戶的初始金額(通常只有 資產 跟 債務 會有初始值. 收入 跟 支出 是不會有初始值的)
如果你沒有設定初始值的話，有一個簡單的方法可以算出你的初始值。例如：你現在的現金有333, 但是累加結算報告顯示你的現金是-100)
  1. 編輯現金帳戶
  2. 設定初始值為433, 由(333 - (-100))算出

多帳本

你可以建立多個帳本，並設定其中一本為'使用中的帳本'. 在每本帳本中，你可以設定帳本的幣別符號(例如,'$','¥'..等)及其位置. 當你建立一個新的帳本後，程式會建立另外一個資料庫來儲存該帳本的帳戶及明細， 而不同帳本的資料是互相隔離不共用的.

當匯出CSV時,可選擇匯出使用中的帳本(details.csv 和 accounts.csv)或是全部帳本(details-'id'.cvs 和 accounts-'id'.csv, 'id' 為帳本的id). 當匯入CSV時,僅能將使用中的帳本(details.csv 和 accounts.csv)或帳本帳戶(accounts.csv)的CSV匯入目前程式使用中的帳本

當你刪除一帳本時，該關聯的帳戶及明細資料庫也會一伴被刪除. 預設帳戶(id=0)及使用中的帳本是不允許刪除的.

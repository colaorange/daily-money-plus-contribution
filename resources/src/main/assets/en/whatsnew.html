What is new

Dear Users,

Due to the long lack of maintenance and difficulty in updating the [Daily Money+], and with plans to release an iOS version in the future, I have developed a new version called [Daily Money One] using a new programming framework. It is now available on Google Play.

The new version supports importing Daily Money+ CSV files, and has fixed the cloud backup feature. Additionally, it offers new features such as split transactions and transfers between account books, and will continue to add new features and fix bugs.

I should mention that I have included some ads in the app to help cover maintenance costs, but I prioritize ensuring that these ads do not interfere with the user experience.

Due to limited resources, I will only be maintaining and updating [Daily Money One] in the future. You can continue using the current version or try the new version to see if it suits your needs and correctly imports your old data.

If you have any questions, please message us in the Facebook community at Daily Money. Subscribing and following will allow you to see the latest development progress through video updates.
